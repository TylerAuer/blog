# Mountains of Code

*Mountains of Code* is a blog about, well, code. I created it when I decided to change careers from education to software development. 

I try to write about topics that I know sort-of-well. That way I'm forced to read and learn more about something.

[![Screenshot of Mountains of Code](https://github.com/TylerAuer/blog/blob/master/snapshot.png)](https://blog.tylerauer.com)

# Technologies

The site is built with [Hugo](https://gohugo.io/), a static-site generator.
